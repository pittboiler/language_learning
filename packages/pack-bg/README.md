# @ll/pack-bg — Bulgarian language pack

**Data only. Zero logic.** Conforms to the `@ll/pack-schema` `LanguagePack` contract, exactly like
`@ll/pack-mk`. The language-agnostic engines in `@ll/core` consume this pack as a parameter — adding
Bulgarian touched **no `packages/core` code** (the Phase-3 generalization test).

## Provenance & trust
All language content (`src/generated.ts`) is **machine-generated by Opus 4.8** from a Language
Profiler teaching profile, then checked by the automated Validator. It is stamped
`confidence:"unreviewed"` and **gated** — it must pass a **native Bulgarian review** before being
served as authoritative. The spot-check queue is `pipeline/output/review-bg.md`.

Regenerate with: `pipeline/node_modules/.bin/tsx pipeline/src/run-batch.ts` (cached profile in
`pipeline/output/profile-bg.json`).

## Shape
`src/index.ts` assembles the `bulgarian` pack: config (id/voice/ASR hints `bg`,`bg-BG`) + the
generated `alphabet`, `phonology`, `grammar`, `vocab`, `scenarios`, `reader`, and hand-authored
(English-instruction) `writingTasks`.
